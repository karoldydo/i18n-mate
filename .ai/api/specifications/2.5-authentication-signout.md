# 2.5 Sign Out

**Endpoint:** `POST /auth/v1/logout`  
**Description:** Invalidate current session  
**Authentication:** Required (Bearer token)

**Success Response (204 No Content)**

**Error Responses:**

**401 Unauthorized:**

```json
{
  "data": null,
  "error": {
    "code": 401,
    "message": "Invalid or expired token"
  }
}
```

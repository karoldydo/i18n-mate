# 2.1 Sign Up

**Endpoint:** `POST /functions/v1/signup` (Edge Function)  
**Description:** Register new user account. Sends verification email via Resend API. **No session created until email verified.**  
**Authentication:** Public

**Request Body:**

```json
{
  "email": "user@example.com",
  "password": "securePassword123"
}
```

**Success Response (200 OK):**

```json
{
  "data": {
    "email": "user@example.com",
    "email_confirmed_at": null,
    "id": "uuid"
  },
  "error": null
}
```

**Note:** Verification email is sent via Resend API with custom HTML template. After clicking verification link, user is redirected to `/email-verified` page.

**Error Responses:**

**400 Bad Request (Invalid email):**

```json
{
  "data": null,
  "error": {
    "code": 400,
    "message": "Invalid email format"
  }
}
```

**400 Bad Request (Weak password):**

```json
{
  "data": null,
  "error": {
    "code": 400,
    "message": "Password must be at least 8 characters long"
  }
}
```

**403 Forbidden (Registration Disabled):**

```json
{
  "data": null,
  "error": {
    "code": 403,
    "message": "Registration is currently disabled. Please contact support for access."
  }
}
```

**409 Conflict (Email Already Exists):**

```json
{
  "data": null,
  "error": {
    "code": 409,
    "message": "An account with this email already exists"
  }
}
```

**Validation:**

- Email: valid format
- Password: minimum 8 characters
